Предсказание спроса на товары сети вкусвилл

Предсказание спроса на товары осуществляется с помощью модели ARIMA. Модели регулярно переобучаются на исторических данных. Для получения результатов необходимо загрузить xlsx файл с информацией о продаже товаров. В файле должны быть следующие данные - id товара, id магазина, цена товара и объем продаж за период На выходе вы получили xlsx файл с объемом закупки товара на следующий период.
